# Deepfake Image Detection System

## Project info

This project is a Vite + React + TypeScript frontend for the Deepfake Image Detection System.

## How can I edit this code?

Use your preferred IDE and run the project locally.

Requirements:
- Node.js
- npm

Steps:

```sh
# Step 1: Install dependencies
npm i

# Step 2: Start the development server
npm run dev
```

## What technologies are used for this project?

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
